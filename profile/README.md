# ママゴトラボ / mamagotolab

**「面倒な業務」を、要件定義から設計してしくみで解決する開発ラボです。**

中小企業・個人事業主の方の「毎月の手作業がつらい」「件数が増えてミスが怖い」を、
Google Apps Script・Python・VBA を使った自動化ツールで解決しています。

---

### 大切にしていること

- **業務の流れを整理してから作る** — プロジェクトマネジメント（PMP）の考え方をベースに、要件定義から設計します
- **事故が起きない設計** — 誤送信・二重処理・入力ミスを前提に、止めるべきところで止まるしくみにします
- **非エンジニアが使える** — 受け取った人がメニューを押すだけで使えるところまで作り込みます
- **動くものを公開する** — 「できます」ではなく、実際に動くテンプレートを置いています

### 公開テンプレート

すぐ業務に使える、動くテンプレートを公開しています。そのまま使うのも、自社向けにカスタマイズするのも自由です。

- 📄 [gas-invoice-automation](https://github.com/mamagotolab/gas-invoice-automation) — 請求書の一括作成＆メール下書き（GAS）
