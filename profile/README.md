# ママゴトラボ / mamagotolab

**「面倒な業務」を、要件定義から設計してしくみで解決する開発ラボです。**

中小企業・個人事業主の方の「毎月の手作業がつらい」「件数が増えてミスが怖い」を、
Google Apps Script・Python・VBA を使った自動化ツールで解決しています。

---

### 大切にしていること

- **業務の流れを整理してから作る** — プロジェクトマネジメント（PMP）の考え方をベースに、要件定義から設計します
- **事故が起きない設計** — 誤送信・二重処理・入力ミスを前提に、止めるべきところで止まるしくみにします
- **非エンジニアが使える** — 受け取った人がメニューを押すだけで使えるところまで作り込みます
- **動くものを公開する** — 「できます」ではなく、実際に動くテンプレートを置いています

### 公開テンプレート

すぐ業務に使える、動くテンプレートを公開しています。そのまま使うのも、自社向けにカスタマイズするのも自由です。

- 📄 [gas-invoice-automation](https://github.com/mamagotolab/gas-invoice-automation) — スプレッドシートから請求書PDFを一括生成し、取引先ごとにGmail下書きまで作る（GAS）
- 📊 [freee-crowdworks-gas](https://github.com/mamagotolab/freee-crowdworks-gas) — クラウドワークスの報酬CSVをfreeeへ自動仕訳（GAS・OAuth2＋Web UI）
- 🔎 [price-watcher](https://github.com/mamagotolab/price-watcher) — 競合価格やニュースなど、指定ページの変化を見張って通知（Python・robots.txt遵守）
- 🧹 [master-checker](https://github.com/mamagotolab/master-checker) — 顧客・商品マスタの重複/空欄/表記ゆれを、自動化を動かす前に一括点検（CLI・pip不要）
